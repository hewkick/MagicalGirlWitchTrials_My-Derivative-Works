### 一、项目说明

本仓库用于存放基于《魔法少女的魔女裁判》的 **NaniNovel** 二创脚本及配套资源。借助 Cursor 的 `.cursor/rules` 提示词，实现从文字稿到脚本的自动化转换，并辅助「中文 → 日文」翻译，减少重复劳动。

使用 @雪莉苹果汁 [B 站链接](https://space.bilibili.com/3546949672241842) 提供的二创工具进行创作。

---

### 二、`.cursor/rules` 规则说明

| 文件 | 用途 |
|:-----|:-----|
| `01_task_discription.mdc` | 文字稿转 NaniNovel 脚本的核心任务描述与格式规范 |
| `02_role_name.mdc` | 人物名称与角色代码对照表 |
| `03_BGM.mdc` | BGM 调用语句对照表 |
| `04_SFX.mdc` | 音效调用语句对照表 |
| `05_background.mdc` | 背景调用语句对照表 |
| `06_Chinese_to_Japanese.mdc` | 中译日翻译规则（角色口癖 + 专有名词 + 易错黑名单） |
| `07_voice.mdc` | 角色配音接入规则（`@voice` 路径、CSV 生成、插入顺序） |

### 一、项目说明

本仓库用于存放基于《魔法少女的魔女审判》的 **NaniNovel** 二创脚本及配套资源。借助 Cursor 的 `.cursor/rules` 提示词，实现从文字稿到脚本的自动化转换，并辅助「中文 → 日文」翻译，减少重复劳动。

- 剧本编辑工具：雪莉的魔审创作工具 @雪莉苹果汁 [【B 站链接】](https://space.bilibili.com/3546949672241842) 

- AI 配音工具：GPT-sovits 2.5 Pro Plus @花儿不哭 [【B 站链接】](https://space.bilibili.com/5760446)

- 魔审角色音色克隆微调模型来源：@神秘捏果女 [【B 站链接】](https://space.bilibili.com/88398063)

- 在创作过程中，我还使用了包括但不限于以下的 AI 工具，特此注明：

    - AICG 生成：Nano Banana Pro
    
    - AI 日语翻译：Claude Sonnet 4.6 (No Thinking)
    
    - AI 翻译一次校对：Gemini 3.0 Pro

    - 处理重复代码编写：Cursor Composer 1.5
    
    - 杂项：豆包 Fast 模型

- 特别感谢：编辑器交流群内各位群友的帮助！！！还有观看视频的大家！！！

### 二、`.cursor/rules` 规则说明（会不断更新与修正）

| 文件 | 用途 |
|:-----|:-----|
| `01_task_discription.mdc` | 文字稿转 NaniNovel 脚本的核心任务描述与格式规范 |
| `02_role_name.mdc` | 人物名称与角色代码对照表 |
| `03_BGM.mdc` | BGM 调用语句对照表 |
| `04_SFX.mdc` | 音效调用语句对照表 |
| `05_background.mdc` | 背景调用语句对照表 |
| `06_Chinese_to_Japanese.mdc` | 中译日翻译规则 |
| `07_voice.mdc` | 角色配音接入规则 |
